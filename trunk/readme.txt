Cata UI Toolkit Instructions:

Place the folder Cata-UI-Toolkit inside your World of Warcraft Beta or
World of Warcraft Public Test folder, and double-click on extract-code.bat
or extract-art.bat to extract the user interface source code or user interface
art files respectively.

You can edit the batch files if you like.  "extract.exe" takes these command
line options:

Path to "World of Warcraft\Data"
  --datadir DIRECTORY

Output directory
  --outdir DIRECTORY

Locale (default enUS)
  --locale LOCALE
  
What to extract (default code)
  --extract code
	--extract art
